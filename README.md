# GuillaumeC-MSFT.github.io

Personal technical blog for Microsoft Purview, Microsoft 365 security, Zero Trust, and information protection guidance.

## Publishing

This site is designed for GitHub Pages with no build tooling required.

1. Create a public GitHub repository named `GuillaumeC-MSFT.github.io`.
2. Push this folder to the repository's `main` branch.
3. In GitHub, go to **Settings > Pages** and publish from the `main` branch root.
4. Optional: add a `CNAME` file containing your custom domain and configure DNS.

## Local preview

Open `index.html` directly in a browser.
